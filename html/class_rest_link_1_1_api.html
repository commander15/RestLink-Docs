Detailed Description

Represents an API interface that manages configuration, requests, and parameters.

The Api class is used to interact with a specific API, including managing the API's name, version, URL, and request parameters. It provides methods for setting and retrieving these properties, as well as configuring the API using both direct URLs and JSON configuration objects. The class allows for adding, removing, and managing request parameters.
