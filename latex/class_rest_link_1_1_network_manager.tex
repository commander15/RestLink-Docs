\doxysubsection{Detailed Description}
A custom network manager for handling HTTP requests and responses in Rest\+Link. 

This class extends QNetwork\+Access\+Manager to provide additional functionality, including custom caching and cookie management, as well as redirect policy control. It is designed to be the core network handling component of Rest\+Link.
