\doxysubsection{Detailed Description}
A custom cookie jar for managing HTTP cookies in Rest\+Link. 

This class extends QNetwork\+Cookie\+Jar to provide custom cookie handling for the Rest\+Link network manager. It allows for advanced control over cookies in the HTTP requests and responses.
