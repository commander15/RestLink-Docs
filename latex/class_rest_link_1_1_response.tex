\doxysubsection{Detailed Description}
Represents a response to an API request in the Rest\+Link framework. 

The \doxylink{class_rest_link_1_1_response}{Response} class encapsulates the response from an API request, providing methods for accessing various aspects of the response such as status, headers, body, and progress. It also provides signals for progress updates, completion, and error handling.

The \doxylink{class_rest_link_1_1_response}{Response} object is associated with a specific API request and provides functionality to handle download/upload progress, SSL errors, network errors, and HTTP status codes. It also supports reading the response body as JSON or raw data and provides the option to abort or ignore specific errors during the process.
