\doxysubsection{Detailed Description}
Represents an API interface that manages configuration, requests, and parameters. 

The \doxylink{class_rest_link_1_1_api}{Api} class is responsible for creating network requests and managing response data. It supports adding parameters to requests, configuring the API from different sources, and emitting signals for configuration changes and network error events.
